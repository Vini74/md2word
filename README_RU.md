# Конвертер Markdown в Word

Локальное веб-приложение для конвертации Markdown в формат, совместимый с Microsoft Word.

## Что это

Одностраничное приложение (SPA), работающее в браузере. Конвертирует Markdown-текст в HTML, который можно скопировать в буфер обмена и вставить в Word с сохранением форматирования.

## Основные функции

- Конвертация Markdown в Word-совместимый HTML с inline-стилями
- **Встраивание изображений** в документ (конвертация в base64 при конвертации)
- **Стили для исходного кода** с сохранением отступов (шрифт Courier New)
- Кнопка «Копировать DOC в буфер» — создаёт данные в формате CF_HTML для корректной вставки в Word
- Предпросмотр результата в реальном времени
- Загрузка Markdown-файлов (.md, .markdown, .txt)
- Скачивание результата как HTML-файла
- Автоматическая конвертация при вводе
- Поддержка сложных структур: многоуровневые списки, таблицы, код, цитаты, изображения, заголовки H1-H6

## Как использовать

1. Откройте `index.html` в браузере или запустите локальный сервер (рекомендуется)
2. Введите Markdown в левое поле или загрузите файл
3. Убедитесь, что включена опция "📊 Стили для Word" (для встраивания изображений)
4. Нажмите «Конвертировать» (или дождитесь авто-конвертации)
5. Нажмите «Копировать DOC в буфер»
6. Вставьте (Ctrl+V) в Microsoft Word

## Технические детали

- Чистый HTML/CSS/JavaScript, без серверной части
- Использует библиотеку Showdown.js (включена локально в `/lib`)
- Clipboard API для работы с буфером обмена
- **Изображения конвертируются в base64** для встраивания в документ (работает без интернета в Word)
- **Код форматируется с сохранением отступов** через замену пробелов на `&nbsp;`
- Формат CF_HTML обеспечивает корректную вставку в Word
- Все данные обрабатываются локально, работает полностью без интернета

## Запуск

### Простой способ (открытие файла)
Просто откройте `index.html` в браузере.

⚠️ **Важно**: При открытии файла напрямую (`file://`) изображения могут не загружаться из-за ограничений CORS.

### Рекомендуемый способ (локальный сервер)
Для корректной загрузки изображений запустите локальный сервер:

```bash
# Python 3
python -m http.server 8000

# С привязкой к IPv4
python -m http.server 8000 --bind 0.0.0.0
```

Или используйте npx:
```bash
npx http-server
```

Затем откройте: `http://localhost:8000`

## Поддерживаемые элементы Markdown

- ✅ Заголовки H1-H6
- ✅ **Жирный текст**, *курсив*, ***жирный курсив***
- ✅ ~~Зачёркнутый текст~~
- ✅ `Инлайн-код` и блоки кода с сохранением отступов
- ✅ [Ссылки](https://example.com)
- ✅ Изображения (встраиваются в документ как base64)
- ✅ Маркированные и нумерованные списки
- ✅ Вложенные списки (4+ уровня)
- ✅ Чекбоксы (task lists)
- ✅ Таблицы
- ✅ Цитаты с вложенностью
- ✅ Горизонтальные разделители
- ✅ Перенос строк (два пробела в конце строки)
- ✅ Экранирование специальных символов

## Ограничения

- Некоторые расширенные возможности таблиц Markdown могут не поддерживаться
- Требуется современный браузер с поддержкой Clipboard API
- Для корректной загрузки изображений рекомендуется использовать локальный сервер
