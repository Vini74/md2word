# Конвертер Markdown в Word

Локальное веб-приложение для конвертации Markdown в формат, совместимый с Microsoft Word.

## Что это

Одностраничное приложение (SPA), работающее в браузере. Конвертирует Markdown-текст в HTML, который можно скопировать в буфер обмена и вставить в Word с сохранением форматирования.

## Основные функции

- Конвертация Markdown в Word-совместимый HTML
- Кнопка «Копировать DOC в буфер» — создаёт данные в формате CF_HTML для корректной вставки в Word
- Предпросмотр результата в реальном времени
- Загрузка Markdown-файлов (.md, .markdown, .txt)
- Скачивание результата как HTML-файла
- Автоматическая конвертация при вводе
- Поддержка сложных структур: многоуровневые списки, таблицы, код, цитаты

## Как использовать

1. Откройте `index.html` в браузере
2. Введите Markdown в левое поле или загрузите файл
3. Нажмите «Конвертировать» (или дождитесь авто-конвертации)
4. Нажмите «Копировать DOC в буфер»
5. Вставьте (Ctrl+V) в Microsoft Word

## Технические детали

- Чистый HTML/CSS/JavaScript, без серверной части
- Использует библиотеку Showdown.js (включена локально в `/lib`)
- Clipboard API для работы с буфером обмена
- Формат CF_HTML обеспечивает корректную вставку в Word
- Все данные обрабатываются локально, работает полностью без интернета

## Файлы проекта

- `index.html` — основное приложение
- `lib/showdown.min.js` — библиотека для конвертации Markdown
- `BigExample.md` — пример сложного Markdown-документа
- `prompt.txt` — исходное ТЗ

## Запуск

Просто откройте `index.html` в браузере. Для лучшей совместимости можно запустить локальный сервер:

```bash
python -m http.server 8000
# или
npx http-server
```

## Ограничения

- Некоторые расширенные возможности таблиц Markdown могут не поддерживаться
- Требуется современный браузер с поддержкой Clipboard API
- Изображения по внешним URL загружаются только при наличии интернет-соединения
