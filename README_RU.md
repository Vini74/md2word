# Конвертер Markdown в Word

Локальное веб-приложение для конвертации Markdown в формат, совместимый с Microsoft Word. Поддерживает работу с одним файлом и пакетную обработку множества файлов с объединением в один документ.

## Что это

Одностраничное приложение (SPA), работающее в браузере. Конвертирует Markdown-текст в HTML, который можно скопировать в буфер обмена и вставить в Word с сохранением форматирования.

## Основные функции

### Работа с одним файлом
- Ввод Markdown в текстовое поле
- Загрузка отдельного файла (.md, .markdown, .txt)
- Предпросмотр результата в реальном времени
- Автоматическая конвертация при вводе

### Пакетная обработка множества файлов
- **Drag & Drop** — перетаскивайте файлы из проводника прямо на панель очереди
- **Кнопка загрузки** — выберите один или несколько файлов через диалог
- **Управление порядком** — кнопки ↑ ↓ для точного позиционирования
- **Сортировка по имени** — быстрая сортировка файлов в алфавитном порядке
- **Автоматическое объединение** — все файлы конвертируются в один большой документ
- **Опция показа имён файлов** — чекбокс для добавления имени файла перед каждым фрагментом
- **Разрыв страниц** — между файлами добавляется разрыв страницы для Word
- **Очистка очереди** — кнопка «🗑️ Очистить очередь» сбрасывает только список файлов

### Общие возможности
- Конвертация Markdown в Word-совместимый HTML с inline-стилями
- **Встраивание изображений** в документ (конвертация в base64 при конвертации)
- **Стили для исходного кода** с сохранением отступов (шрифт Courier New)
- Кнопка «Копировать DOC в буфер» — для корректной вставки в Word
- Кнопка «Копировать HTML в буфер» — для вставки чистого HTML
- Скачивание результата как HTML-файла
- Поддержка сложных структур: многоуровневые списки, таблицы, код, цитаты, изображения, заголовки H1-H6
- **Toast-уведомления** — всплывающие сообщения о действиях (10 сек)
- **Статус-бар** — постоянное отображение текущего состояния

## Как использовать

### Один файл
1. Откройте `index.html` в браузере или запустите локальный сервер (рекомендуется)
2. Введите Markdown в текстовое поле или загрузите файл кнопкой «📂 Загрузить Markdown»
3. Убедитесь, что включена опция «📊 Стили для Word» (для встраивания изображений)
4. Нажмите «Конвертировать» (или дождитесь авто-конвертации)
5. Нажмите «📋 Копировать DOC в буфер»
6. Вставьте (Ctrl+V) в Microsoft Word

### Множество файлов
1. Откройте `index.html` в браузере
2. **Перетащите файлы** из проводника на панель «📑 Очередь файлов» **ИЛИ** нажмите «📂 Загрузить Markdown» и выберите несколько файлов
3. Файлы автоматически объединятся и сконвертируются
4. **Измените порядок** кнопками ↑ ↓ (конвертация выполнится автоматически)
5. При необходимости включите чекбокс «📄 Показывать имя файла» для добавления заголовков
6. Нажмите «📋 Копировать DOC в буфер» и вставьте в Word

### Управление очередью файлов
| Действие | Описание |
|----------|----------|
| ↑ ↓ | Переместить файл вверх/вниз |
| ✕ | Удалить файл из очереди |
| 🔤 Сортировать по имени | Алфавитная сортировка |
| ➕ Добавить ещё файлы | Добавить файлы к существующим |
| 🗑️ Очистить очередь | Сбросить только список файлов |
| 🗑️ Очистить (вверху) | Сбросить всё (редактор + очередь) |

## Технические детали

- Чистый HTML/CSS/JavaScript, без серверной части
- Использует библиотеку Showdown.js (включена локально в `/lib`)
- Clipboard API для работы с буфером обмена
- **Изображения конвертируются в base64** для встраивания в документ (работает без интернета в Word)
- **Код форматируется с сохранением отступов** через замену пробелов на `&nbsp;`
- Формат CF_HTML обеспечивает корректную вставку в Word
- Все данные обрабатываются локально, работает полностью без интернета

## Запуск

### Простой способ (открытие файла)
Просто откройте `index.html` в браузере.

⚠️ **Важно**: При открытии файла напрямую (`file://`) изображения могут не загружаться из-за ограничений CORS.

### Рекомендуемый способ (локальный сервер)
Для корректной загрузки изображений запустите локальный сервер:

```bash
# Python 3
python -m http.server 8000

# С привязкой к IPv4
python -m http.server 8000 --bind 0.0.0.0
```

Или используйте npx:
```bash
npx http-server
```

Затем откройте: `http://localhost:8000`

## Поддерживаемые элементы Markdown

- ✅ Заголовки H1-H6
- ✅ **Жирный текст**, *курсив*, ***жирный курсив***
- ✅ ~~Зачёркнутый текст~~
- ✅ `Инлайн-код` и блоки кода с сохранением отступов
- ✅ [Ссылки](https://example.com)
- ✅ Изображения (встраиваются в документ как base64)
- ✅ Маркированные и нумерованные списки
- ✅ Вложенные списки (4+ уровня)
- ✅ Чекбоксы (task lists)
- ✅ Таблицы
- ✅ Цитаты с вложенностью
- ✅ Горизонтальные разделители
- ✅ Перенос строк (два пробела в конце строки)
- ✅ Экранирование специальных символов

## Ограничения

- Некоторые расширенные возможности таблиц Markdown могут не поддерживаться
- Требуется современный браузер с поддержкой Clipboard API и Drag & Drop
- Для корректной загрузки изображений рекомендуется использовать локальный сервер
