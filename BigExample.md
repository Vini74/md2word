### Важные замечания по структуре Markdown-списков

- Каждый новый уровень вложенности создаётся отступом в **3 пробела** или **табуляцией**.
- В нумерованных списках можно писать любые числа (например, `1`, `5`, `10`), но отображаться они будут по порядку: `1, 2, 3...`.
- Для разрыва между абзацами внутри элемента списка нужно делать отступ перед пустой строкой.
- Для заголовков используются символы `#`, `##`, `###` и т.д.
- Для выделения текста: **жирный** (*две звёздочки*), *курсив* (*одна звёздочка*).
- Для кода: `` `код` ``.
- Для ссылок: `[текст](url)`.
- Для изображений: `![alt-текст](url)`.
- Для горизонтальной линии: `---`.

Если нужен шаблон под конкретную задачу (например, техническое задание, план проекта, инструкция), уточните детали — подготовлю структуру под ваши цели!
